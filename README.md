# 📰 College Blog Project

Welcome to the **College Blog Project**! This is a dynamic News Blogging Website specifically designed for college students to read, write, and share interesting news and academic articles.

---

## 🚀 Features

* **User Authentication:** Secure Login and Registration system for students.
* **Content Management:** Options to easily create (write) and delete blog posts.
* **Interactive UI:** Smooth toggling between **Dark Mode** and **Light Mode** for a better reading experience.

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript (or React.js if used)
* **Backend:** Node.js (if used for server-side logic)
* **Database:** MongoDB / MySQL (if used to store blogs)

---

## 💻 How to Run the Project Local Layout

Follow these simple steps to run this project on your local computer:

1. **Download the Project:** Clone or download this repository to your local machine.
2. **Navigate to the Directory:** Open the project folder in your favorite code editor (like VS Code).
3. **Launch the App:** Open the `index.html` file directly in any modern web browser (Chrome, Edge, Safari).
